Listas ordenadas (<ol>)

Vamos aqui primeiro do jeito clássico, isto é, inserido uma por uma. Lembrando que para listar, usar a tag <li>

Parece que podemos usar style dentro da tag, ou usar CSS na head com style - ol { list-style-type: lowe/upper-greek-latin-alpha;}. W3 referencia essa última.

    Coisas que fazer

  1. Acordar
  2. Abrir o olho
  3. Falar ahhhh
  4. Pegar o celular
  5. ahh
  6. uui

Para já fazer um número pré determinado, digitar ol>li*5. Lmbrando que não é obrigatório o fechamento no <li>, mas no ol sim.

  1. ah
  2. ta
  3. sla
  4. sla
  5. sla
  6. sla
  7. nehhh

Parece que dentro da tag no body, se pode colocar class="a", b ou outra, para no CSS, qunado quiser especificar, digitar apenas ol.a, e não modificar todas. Interessante.
